# Midnight Glitch Explorer

A fun, cyberpunk-themed viewer for Midnight testnet transactions.

## Setup
1. Add your Blockfrost key to Vercel env vars (REACT_APP_BLOCKFROST_KEY).
2. Deploy to Vercel.
3. When mainnet live: Change 'midnighttestnet' to 'mainnet' in App.js.

## Features
- Live block fetches
- Glitch effects & confetti
- Easy switch to mainnet
